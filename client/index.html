To run the app

* Console 1: run "npm install" * Console 2 (optional): run "seed", if you wish to use the sample data provided * Console 3: run "mongod" * Console 4: run "npm start" in root directory * Enter url "http//localhost:5500"

Functionality:

1. Add a burger to the DB via .post request

2. https://localhost:5500/burgers/random select a random burger

3. https://localhost:5500/burgers get all burgers

4. https://localhost:5500/burgers?name=BURGERNAME get a burger by name

5. https://localhost:5500/burgers?page=2&per_page=3 pagination
